Рекламный баннер на Vue

## Инструкция по установке
Важно!!! Разработка баннера велась на ОС Windows 10 и отладка в Google Chrome v77.0.3865.120
```
$ cd kvadroom-master
$ npm install
$ npm run dev
```
## Использовалось
+ PUG
+ Stylus
+ Bootstrap (только в целях .container)
+ jQuery
+ VueJS
+ Flexbox
